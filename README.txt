Helps to organize columns in the WordPress admin area, you can add columns from meta values and set the default sort column.

== Description ==

For the first version of this plugin the WordPress Users columns were addressed, so you can add more columns there like User ID or the date the User Registered. Additionally, you can add WordPress user meta or BuddyPress XProfile meta in as extra columns for quick viewing of important metadata. 

== Installation ==

1. Download the zip file and install like a normal plugin. 
2. Adjust the options under Users -> Columns Organizer
